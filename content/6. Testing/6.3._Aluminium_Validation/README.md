To confirm if the material and simulation is a problem, aluminium specimens were used to validate the 3 point bend tests. This removes the complexities of Ansys ACP and provides a material with known properties which should result in a more accurate simulation.

As there is no specific ASTM standard for doing 3 point bending on aluminium pieces, the aluminium specimens are cut to the same dimensions of the carbon fiber ones for ease of simulation. Testing showed the same xx% error as seen in the carbon fiber simulations. Since the material is isotropic, the theorethical deflection is calculated and it is found that the theorethical and simulated deflections are closer to each other than the experimental values. This points in the direction that the experimental values are incorrect.
