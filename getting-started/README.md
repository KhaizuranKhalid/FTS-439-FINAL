# Getting Started

Welcome to the NUS Formula SAE (FSAE) Powertrain documentation. NUS FSAE is a student team from the National University of Singapore that designs, builds, and races an electric Formula-style car in the Formula SAE competition. This site serves as the central documentation for the Powertrain subteam: architecture, design rationale, testing, operations, and maintenance. Learn more about the team on the official site: [NUS Formula SAE](https://www.nusformulasae.com/).
